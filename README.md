# 🎬 VideoTemplate

A modern AI-powered video template platform built with Next.js, Supabase, and advanced frontend animations.

## 🚀 Features

- 🔐 Authentication System
- 🎨 Modern UI/UX
- ⚡ Next.js App Router
- 🗄️ Supabase Integration
- 📱 Fully Responsive Design
- ✨ Smooth Animations
- 🌙 Dark Mode UI
- 🔥 Optimized Performance

---

## 🛠️ Tech Stack

- Next.js 15
- React
- Tailwind CSS
- Supabase
- TypeScript
- Framer Motion
